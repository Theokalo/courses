Υλικό μαθήματος προγραμματισμού (C++)¶

Περιεχόμενα:

  • Ενότητα 2-1
    • Native data types: char, int, double [και bool]
    • Αυτόματες μετατροπές (Implicit conversions)
    • Τελεστές (Operators)
    • Παράδειγμα: calculator
  • Ενότητα 2-2
    • Ανίχνευση σφαλμάτων Εισόδου-Εξόδου (I/O Errors)
    • Χειρισμός σφαλμάτων στο stream cin
  • Ενότητα 3-1
    • Παράδειγμα: Calculator με χειρισμό σφαλμάτων εισόδου
    • Εισαγωγή στις συναρτήσεις
  • Ενότητα 3-2
    • Πώς και τι περνά ως όρισμα σε μια συνάρτηση
    • Πέρασμα παραμέτρων by reference
    • By value ή by reference;
    • Const references
    • Εμβέλεια (Scope)
    • Τι επιστρέφει μια συνάρτηση
  • Ενότητα 4-1
    • Εισαγωγή στις κλάσεις
    • Ανάθεση αρχικών τιμών στα μέλη της κλάσης
    • Προσπέλαση μελών ενός αντικειμένου
  • Ενότητα 4-2
    • Στατικές μεταβλητές-μέλη κλάσης
    • Κληρονομικότητα - βασικές οδηγίες
    • Ένα απλό παράδειγμα
  • Ενότητα 5-1
    • Αρχείο επικεφαλίδων (header file)
    • Αρχείο που περιέχει τη main()
    • Αρχείο υλοποίησης (implementation file)
    • Παραγωγή εκτελέσιμου κώδικα
  • Ενότητα 5-2
    • Χειρισμός σφαλμάτων (Exceptions)
    • Υπερφόρτωση τελεστών (operator overloading)
    • Default constructor
    • Διαβάζοντας από αρχείο (formatted input)
